# Change Log

> **Purpose**: Track every documentation update — who changed what, and when.
> This is the single source of truth for vault activity.

---

## AI Rule — Mandatory Change Log Entry

> **RULE**: Every time a document in this vault is created, updated, or deleted,
> a new row **must** be added to the table below **before** the change is considered complete.
>
> Fields:
> - **Date** — `YYYY-MM-DD`
> - **Who** — Name or identifier of the person / AI making the change
> - **Document** — Obsidian `[[wikilink]]` to the affected document
> - **Action** — `Created` / `Updated` / `Deleted`
> - **Summary** — One-line description of what changed
>
> When the user asks to **"synchronize"** or **"do an entry"**, this means:
> add a row to this table reflecting the work done.
