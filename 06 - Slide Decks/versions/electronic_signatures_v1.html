Where You Can Use Signatures

Three integration levels — pick what you need
Ad-hoc documents

Any PDF

Upload contracts, NDAs, agreements, policies — anything. Design the template once, reuse it.

Always available

This replaces 100% of basic DocuSign usage.

Purchase orders

PO Signature

From any PO: Actions → Request Signature. The signed PO attaches to the record automatically.

Optional — you decide

Useful for formal supplier acknowledgements.

Sale orders

SO / Quotation Signature

Option A — Customer signs on the portal page (lightweight).
Option B — Full Sign template sent by email (formal).

Optional — you decide

Option A needs no Sign module — built into Sales.

Bonus: In Odoo 19, you can request a signature from any Odoo record via the gear icon menu — invoices, CRM opportunities, HR contracts, and more.
