Overview

What is EXCO Projects?

An executive view of every strategic initiative running across Kebony

THE PROBLEM

  • What strategic projects are active, allocated, stalled or done
  • Who owns each one and what domain it covers
  • Which ones are overdue, at risk, or coming up

Spreadsheets and emails don't give that. Projects get tracked 12 different ways across 12 owners.

THE SOLUTION

EXCO Projects gives:

  • Dashboard — KPIs + upcoming deadlines
  • Project Portfolio — kanban of every strategic project, grouped by stage
  • Gantt — timeline view across all projects

Each project has an owner who manages tasks underneath. The portfolio updates automatically as task work progresses.

Mental model: Two levels — portfolio (projects on a kanban) and execution (tasks inside a project). The portfolio auto-updates as task activity happens.
📋

10 PRECONFIGURED DOMAINS

Finance · Production · Supply Chain · Quality · Certification · Sales · Marketing · US Market · IT · Odoo

Fully customizable — add, rename, or disable via Configuration.

The EXCO Dashboard

Your home screen · 4 KPIs + upcoming deadlines · refreshes on open
01-dashboard.png EXCO Dashboard with 4 KPI tiles:
In Progress · CEO Inbox · Active Total · Overdue
+ "Upcoming Deadlines" card below

THE 4 KPI TILES

  • In Progress — projects where work is actually happening (at least one task moved to "Ongoing")
  • CEO Inbox (Non-Allocated) — projects created but not yet assigned to an owner. The action list for allocation.
  • Active Total — everything not done (overview of total load)
  • Overdue — projects past their planned end date, highlighted red
Tip: Click any tile to drill into the portfolio kanban filtered to just those projects.

UPCOMING DEADLINES

Shows the 8 projects closest to their planned end date in the next 30 days. Name, owner, domain, due date. A "what to worry about this month" at-a-glance.

When to use it: Start your morning here. 30 seconds of KPIs tells you if anything changed overnight — new overdue? New project in your inbox? Then move on.

Project Portfolio

The Portfolio Kanban

Every strategic project, grouped by stage · drag to move · click a name to zoom in
02-portfolio-kanban.png Kanban with 4 columns:
Non-Allocated · Allocated · In Progress · Done
Each card = one project

THE 4 PORTFOLIO STAGES

  • Non-Allocated — Project created but no owner yet
  • Allocated — Owner assigned; no task progress yet
  • In Progress — First task moved to "Ongoing" — work is live
  • Done — All tasks finished
Auto-transition: Cards move automatically as task activity happens. The kanban is a read-out, not a manual tracker.

HOW TO NAVIGATE

  • Click a project name → zooms into its tasks
  • Click elsewhere on the card → opens project form
  • "New" top-left → create a new project (auto-flagged EXCO)

Create a project

Creating a new EXCO project

4 fields, 30 seconds · fill them, save, assign
03-project-form.png Project form with Portfolio section showing:
EXCO Project toggle · Domain · Priority
+ Planned Date (native Odoo range)

FIELDS YOU FILL

Project name
What it's called, e.g. "Q2 US marketing plan"
Project Manager
The owner. Usually a C-team member or senior lead.
Domain
One of the 10 (Finance, IT…). Drives reporting.
Priority
Low / Medium / High — shows as stars on the card.
Planned Date
Start → End range. Used by Gantt + "Overdue" KPI.
EXCO Project
Toggle (already ON when created here). Marks it visible in the EXCO portfolio.
Quick path: Portfolio → New → name → Domain + Priority → Planned Date → Project Manager → save (☁). Without a Project Manager, the project sits in Non-Allocated. Once an owner is set, it auto-moves to Allocated.
Tip — Tasks smart button: Once saved, the project form has a Tasks button at the top (shows "0 / 0 (0%)" when empty). Click it to jump straight into the task kanban for that project — fastest path from project form to the tasks view.

Zooming into a project

04-task-view.png Task kanban inside a specific EXCO project.

WHAT YOU SEE

The tasks that make up this project. Owner + team break the project into actionable tasks here. Useful to check progress when needed.

TASK STAGES

Non-Allocated
Ongoing
Done
The magic: When the first task moves to Ongoing, the parent project auto-moves to "In Progress" on the EXCO portfolio kanban. When all tasks are Done, the project moves to Done. No manual updates needed.

NAVIGATION — WHEN IN A PROJECT

⬅️ GO BACK

Click "EXCO Projects" in the breadcrumb to return to the portfolio.

➕ NEW TASK

Click "+" in any stage column. Creates a task within this project.

📊 TASK GANTT

Click the Gantt icon in the top-right view switcher — shows this project's tasks on a timeline.

⚙️ EDIT PROJECT

Click the gear ⚙ next to the title to reopen the project form.

Gantt

The Portfolio Gantt

Timeline view · projects plotted as bars · grouped by domain · colored by stage
05-gantt.png Gantt timeline with projects as bars,
grouped by domain in swim lanes.
Today column highlighted.

WHAT IT SHOWS

  • One bar = one project, from its Planned Start to Planned End
  • Swim lanes by domain — all Finance projects together, all IT projects together, etc.
  • Bar color = portfolio stage (so you can see at a glance which are In Progress vs Done)
  • Today is highlighted — easy to see what's late vs upcoming

WHEN TO USE IT

  • End-of-month portfolio review
  • Capacity check: "Is IT overloaded vs Finance?"
  • Planning: "What's my Q3 workload by domain?"
  • SteerCo prep — screenshot the Gantt straight into the deck
Requires dates: Projects without a Planned Start / End don't show on the Gantt. Make sure you fill those when creating.
EXCO Projects · CEO User Guide 7 / 7
Cheat sheet

Quick reference

Print this, stick it on your monitor · 30-second refresher

DAILY ROUTINE (2 MINUTES)

  1. Open EXCO Projects → Dashboard
  2. Check the 4 KPIs — any new Overdue? Any unallocated in CEO Inbox?
  3. Skim Upcoming Deadlines — anything hitting in the next 2 weeks?
  4. If action needed: click the KPI tile → drill into the list → act (assign, call, etc.)

WEEKLY ROUTINE (10 MINUTES)

  1. Open Project Portfolio — review every column
  2. Non-Allocated — assign owners
  3. Allocated — nudge owners that haven't started
  4. In Progress — check nothing's stuck
  5. Open Gantt — confirm no capacity crunch in any domain

KEY TERMS

EXCO Project — a strategic initiative tracked at C-level.

Domain — the business area a project belongs to (Finance, IT, etc).

Owner (Project Manager) — the person accountable for delivering the project.

WHAT YOU DO

✅ Create projects
✅ Assign owners
✅ Set domain + priority + dates
✅ Check portfolio health
✅ Use Gantt for planning

You do NOT manage individual tasks — that's the owner's job.
